## Código Fuente

Los archivos principales del proyecto son:

- `procesamiento_en_BATCH.py`: Procesamiento en batch con Spark.
- `productor.py`: Simulación de datos en tiempo real.
- `spark_streaming_consumidor.py`: Procesamiento en streaming con Spark.

Cada archivo contiene la implementación correspondiente a cada etapa del proyecto, con el fin de Identificar y predecir los niveles de incidencia de violencia de género en las comunas de Villavicencio utilizando variables demográficas y temporales.
